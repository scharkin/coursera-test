Welcome to AngularJS.

Author page content using GitHub Flavored Markdown, select a template crafted by a designer, and publish. After page is generated, check out the new gh-pages branch locally.

Designer Templates

For templates click 'Continue to layouts' to browse through them. Edit page before publishing. After publishing page, revisit the page generator and switch to another theme. Page content will be preserved.

Creating pages manually

Push branch gh-pages to local repository. In addition to supporting regular HTML content, GitHub Pages support Jekyll, a simple, blog aware static site generator. Jekyll makes it easy to create site-wide headers and footers without having to copy them across every page. It also offers intelligent blog support and other advanced templating features.
